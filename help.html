How does Ihurem work?

Magic or super powers... Not. Ihurem is powered by Javascript.


const urlInput = document.getElementById('url-input');
const displayButton = document.getElementById('display-button'); const iframeArea = document.getElementById('iframe-area');

It takes an HTML element and defines it as a variable.


const blacklist = [
'https://proxysite.com',
'https://www.croxyproxy.rocks/_ja/',
'https://proxyium.com/ja/',
'https://www.blockaway.net/_ja/'
];

This defines the blacklist.


displayButton.addEventListener('click', () => {
const url = urlInput.value;

if (isValidURL(url) && !isBlacklisted(url)) {
iframeArea.src = url;
} else {
alert('Invalid URL or blacklisted.');
}
});

Get the URL entered in the text box when the “Show” button is clicked.
If the URL is incorrect or blacklisted, an error message is displayed; if the URL is correct, a web page with that URL is displayed in the iframe.
