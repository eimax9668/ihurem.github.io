About Ihurem Help

Introduction

You may have found that when you tried to visit a site related to your studies at school or at home, it was blocked by filtering.
But it doesn't have to be that way. Ihurem is a website that displays the sites you want to see in an iframe. Ihurem could be the solution to your unreasonable filtering problems.

How it works

ihurem is controlled by an HTML element called iframe.
<Iframe> is an HTML element that represents a nested viewing context and allows other pages to be embedded within the current HTML page.

Use Ihurem

Ihurem supports both online and offline usage.
Want to customize Ihurem? You can download the source from GitHub and build your own Ihurem.
